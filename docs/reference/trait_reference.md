# Catalogo label e descrizioni approvate per i Trait

Usa questa pagina come catalogo condiviso di etichette e descrizioni già validate. Quando aggiungi un nuovo trait, prova prima a riutilizzare uno dei valori esistenti o proponi un aggiornamento qui.

## Voci consigliate
| Slug suggerito | Label approvata | Descrizione approvata |
| --- | --- | --- |
| `body-length` | Lunghezza corporea | Misura della lunghezza totale dell'esemplare adulto in centimetri o metri. |
| `diet` | Dieta principale | Categoria alimentare prevalente (erbivoro, carnivoro, onnivoro, detritivoro, ecc.). |
| `activity-cycle` | Ciclo di attività | Finestra temporale in cui la specie è più attiva (diurna, notturna, crepuscolare). |
| `habitat` | Habitat dominante | Ambiente principale in cui la specie vive o nidifica. |

## Come proporre nuove voci
1. Aggiungi una riga alla tabella con **slug**, **label** e **descrizione** concise.
2. Mantieni coerenza lessicale con le voci esistenti (singolare, maiuscole/minuscole, uso di trattini nello slug).
3. Se la voce viene approvata, aggiorna le schede/guide che la utilizzano.
